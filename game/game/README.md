Проект создан для выдуманной сети компьютерных клубов "Шрек"

**Стек**: HTML, CSS, JavaScript, PHP

**Структура проекта**

- папка images - содержит картинки

- папка pages - содержит html размсетку сайта
- -  login.html - вход в систему
- -  main.html - основная страница сайта, где описана информация о клубе
- -  news.html - страница с новостями
- -  profile.html - профиль пользователя
- - tariffs.html - страница с описание тарифов

- папка scripts - содержит скрипты для рабочего сайта
- - checkUser.js - проверка пользователя
- - news.js - модальные окна новостей
- - profile.js - подтягивание данных из формы для заполнения, чтобы к каждому пользователю была уникальная информация
- - tariffs.js - модальные окна тарифов
- - theme.ts - смена темы

- папка styles - содержит стили
- - style.css - все стили в 1 файле

- send_mail.php - для отправки письма с сайта на почту
